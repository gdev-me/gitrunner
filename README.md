# 📌 About

GitRunner is an open-source platform that analyzes runnable GitHub repositories and automatically converts them into mobile-friendly web applications.

The project combines ideas from:

- GitHub
- Replit
- Hugging Face Spaces
- Docker Playground

while remaining:

✅ Open Source  
✅ Self Hostable  
✅ Mobile First  
✅ AI Assisted  
✅ Plugin Based

---

# ✨ Features

## Repository Analysis

- GitHub URL Import
- Repository Classification
- Language Detection
- Dependency Detection
- Entry File Detection

---

## Runtime Generation

- Python Support
- Node.js Support
- CLI Applications
- Automatic Runtime Configuration

---

## Automatic UI Generation

Generate:

- Forms
- File Uploads
- Buttons
- Log Viewers
- Dynamic Inputs

without writing frontend code manually.

---

## AI Layer

Supports:

- Google Gemini
- OpenAI Compatible APIs
- OpenRouter
- Ollama
- vLLM
- llama.cpp

AI is optional.

---

## Security

Repositories are executed inside:

- Rootless Docker
- Resource Limits
- Read Only Filesystem
- Temporary Containers
- Network Restrictions
